## Run Locally

Clone the project

```bash
  https://github.com/Jahid0178/digi-koein-task.git
```

Go to the project directory

```bash
  cd digi-koein-task
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm run dev
```
